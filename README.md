# DesarrolloWeb-Proyecto

Landing page MVP para **Amor y Sanación** — emprendimiento de bienestar integral en San Andrés, Gral. San Martín, Buenos Aires.

## Marca

- **Nombre:** Amor y Sanación
- **Tagline:** "Un espacio para reconectar con tu bienestar"
- **Instagram:** [@_amor_y_sanacion](https://instagram.com/_amor_y_sanacion)
- **Servicios:** Masajes terapéuticos, terapias holísticas, experiencias sensoriales, contenido de bienestar, acompañamiento emocional

## Stack

HTML + CSS + JS vanilla. Sin frameworks, sin build tools.
